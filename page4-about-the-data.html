ABOUT THE DATA

For this project, I conducted an analysis on a call center dataset recorded between June 1, 2021, and July 1, 2021. The dataset, provided in CSV format, encompasses several columns that include:

  • agent_id: Unique identifier for each agent
  • date: Date of the call
  • product_id: Identifier for the product associated with the call
  • calls_handled: Number of calls handled by each agent on a particular date
  • avg_ht (in seconds): Average time taken to handle a call by each agent
  • avg_queue_time (in seconds): Average time spent by a call in the queue before being answered
  • Customer_satisfaction: Customer satisfaction score associated with the call

In addition to the call center dataset, there are two reference tables available. The first table consists of five columns, which are:

  • Agent_ID Unique identifier for each agent
  • First_Name First name of the agent
  • Last_Name Last name of the agent
  • Hire_Date Date of the agent's employment
  • Supervisor Name of the agent's supervisor

The second table contains the following columns:

  • Product_ID Unique identifier for each product
  • Product_Name Name of the product
  • Call_Review Review or feedback associated with the product

These reference tables will be instrumental in augmenting the call center dataset with additional columns for more comprehensive analysis and insights.
